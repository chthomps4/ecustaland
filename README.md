# Ecusta Land

Public-facing website for Ecusta Land / ecustaland.org.

This site presents an evidence-first civic concept for the former Ecusta mill property in Brevard / Pisgah Forest, North Carolina.

## Purpose

The site is built around:

- transparent brownfield history
- public records and evidence labels
- data-center regulation concerns
- land repair, native planting, trails, and education
- community-scale reuse and public benefit

## Messaging rules

Do not claim that sunflowers or native plantings can fully remediate the Ecusta site. Use careful language:

> Sunflowers and native plantings may support phytomanagement, demonstration planting, habitat repair, soil stabilization, and public education, while cleanup decisions remain subordinate to tract-specific sampling, brownfields restrictions, and agency-approved remediation.

Do not claim that a data center has been approved unless official Ecusta-specific approvals are located.
